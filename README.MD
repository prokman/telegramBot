# Telegram бот для фармацевтического канала
## Параметры сборки и используемые зависимости:
- Java 21;
- Cистема сборки проекта - Maven 3 (Version 3.3.4);
- Spring boot 3.5.6.

## Особенности реализации:
- Поиск по всем постам канала;
- Помощник формирования аптечки;  
- Создание персональных напоминаний, которые будут приходит в виде сообщения от бота;
- Квиз на фармацевтическую тему.
